データ上ではあんなに鮮やかだったのに、印刷物が届いたらなんか色がくすんでる…。そんな経験、ありませんか？

「印刷所のせい？」「用紙の問題？」と思いがちですが、実は原因のほとんどはデータ側にあります。しかも、気づかないうちに踏んでいる「あるミス」が元凶です。

この記事では、印刷で色がくすむ原因と事前に防ぐ方法をデザイナー目線で解説します。CanvaユーザーにもIllustratorユーザーにも対応した内容です。

結論：くすみの原因はRGB→CMYK変換時の「色域の差」

印刷で色がくすむ最大の原因は、RGBのデータをCMYKに変換したときに、モニターで表現できた色が印刷では再現できないことにあります。

モニターはRGB（光の三原色）で色を表現します。一方、印刷はCMYK（インクの四色）で色を再現します。この2つは色の作り方がまったく異なるため、そのまま変換すると色が変わってしまうのです。

特に、モニターで表示できる色の範囲（色域）は印刷よりも広いため、「モニターでは出せるけど印刷では出せない色」が存在します。その部分が変換時に強制的に近似色に置き換えられ、結果としてくすんで見えるという現象が起きます。

RGBとCMYKの基礎から理解したい方は、先に「RGBとCMYKの違いとは？」を読んでおくとスムーズです。

なぜRGBをそのまま印刷すると色がくすむのか

RGBは「光」、CMYKは「インク」

まずイメージをつかんでおきましょう。RGBは光を足して色を作ります。赤・緑・青を全部足すと白になる「加法混色」です。

対してCMYKはインクを重ねて色を作ります。シアン・マゼンタ・イエロー・ブラックを混ぜると暗くなっていく「減法混色」。この2つは色の作り方がそもそも違うため、互いに変換すると必ず何らかの誤差が出ます。

項目RGBCMYK
用途モニター・Web・動画印刷全般
色の作り方光を足して明るくするインクを重ねて暗くする
再現できる色の数多い（色域が広い）少ない（色域が狭い）
変換すると—一部の鮮やかな色が再現できずくすむ

「変換」のタイミングで何が起きているのか

IllustratorやPhotoshopでRGBモードのまま作ったデータを印刷所に入稿すると、印刷所のRIPという機械が自動でCMYKに変換します。このとき、自動変換がかかるわけです。

問題は、この自動変換が「なるべく近い色で」変換してくれるものの、完全に一致する保証はないこと。特に鮮やかな色は再現できる限界を超えているため、くすんだ色に落ち着かされてしまいます。

NOTE

現場のコツ｜入稿前に自分でCMYKに変換しておくと、「どう変わるか」を事前に確認できます。変換結果が想像と違う場合は、CMYKの数値を手動で調整する余地が生まれます。丸投げ変換より圧倒的に仕上がりをコントロールできます。

特にくすみやすい色ワースト3

経験上、次の色は変換後に「あれ？」となることが多いです。

  • 鮮やかな青（#0000FFなど）RGBの青は印刷インクで再現できない領域の代表格。変換すると明らかに暗くくすんだ青になります。コーポレートカラーに青を使っている案件は要注意。
  • ビビッドなオレンジ・ピンクSNSやWebで使われがちな蛍光っぽい色は、印刷すると一気にトーンダウンします。「あの映えオレンジが茶色っぽく…」という悲劇はよく起こります。
  • 彩度の高い緑（草・植物系）自然な緑は変換後にくすんで「疲れた葉っぱ感」が出ることがあります。エコ系・オーガニック系のデザインで多発するパターンです。

Canvaで作ったチラシが印刷でくすんだ場合

最近は「Canvaでチラシを作って、ネット印刷に入稿した」という方も増えています。「画面上ではきれいだったのに、届いたら色がくすんでいた」という声もよく聞きます。

これも原因はまったく同じ。CanvaはデフォルトでRGBを使っています。画面表示に最適化された設計なので、そのまま印刷するとモニターと色が変わりやすいのです。

Canvaのカラーモードを確認する

Canvaには印刷向けに「CMYKでダウンロード」できる機能がありますが、これはCanva Proの有料機能です。無料プランでは通常のRGBのPDFしか書き出せません。

NOTE

Canvaユーザーの選択肢｜Canva Proの場合：「PDFをダウンロード」→「印刷用PDF（CMYK）」を選ぶ。これだけで色がくすむリスクを大幅に減らせます。Canva無料プランの場合：RGBのままPDFを書き出すしかありません。印刷前にCMYKの変換結果を確認しておくと、仕上がりのギャップを事前に把握できます。

Canvaで使いがちな「くすみやすい色」に注意

Canvaのカラーパレットには彩度の高い色が多く並んでいます。画面上での見栄えを重視した設計なので、印刷するとくすみが目立ちやすい傾向があります。

Canvaでよく使われる色印刷後の傾向対策
鮮やかなコーラルピンク・ビビッドオレンジくすんで茶系に寄りやすいM・Yを少し上げてCMYKで調整
発色の良い水色・スカイブルー暗く濁った青になりやすいCとKを控えめにして明るさを確保
テンプレートのグラデーショングラデーションの境目が段になることも印刷用テンプレートか単色ベースを選ぶ

Canvaで印刷物を作るなら、テンプレートを選ぶ段階から落ち着いたカラーを使うのが一番の近道。派手な色は画面映えはするが、印刷では苦しくなりがち。

よくある失敗パターンと注意点

Illustratorのカラーモードをチェックしていない

IllustratorはファイルのカラーモードがRGBかCMYKかを選べます。「ファイル」→「ドキュメントのカラーモード」で確認できますが、新規作成時のデフォルトがRGBになっていることも多く、気づかずに制作してしまうケースがよくあります。印刷物を作るときは最初にCMYKを選ぶのが基本です。

WebからコピーしたカラーコードをそのままIllustratorに貼る

ブランドガイドラインのWebページに載っているHEXコードをそのままIllustratorに入力するのは危険です。そのHEXはRGB用の値なので、CMYKに手動変換する作業がワンステップ必要です。変換せずに使うと、色が意図通りにならない原因になります。

モニターの設定に気づかず「モニターが正しい」と思い込む

モニターの輝度や色温度の設定によって、同じデータでも見え方が変わります。明るめのモニターで作業していると、印刷物が暗く感じやすいという落とし穴があります。できれば印刷所の出力サンプルと見比べる習慣をつけましょう。

NOTE

「色校正（色校）」を依頼すると、本印刷前に実際の印刷物に近いサンプルを確認できます。色にこだわる案件や大量部数の印刷では、コスト以上の安心感があります。

まとめ

印刷で色がくすむのは、RGBとCMYKの色域の違いが原因。事前にCMYKを確認する習慣をつけるだけで防げる。

  1. RGBとCMYKは色の作り方がそもそも違う

    モニターは光、印刷はインク。変換時に再現できない色がくすんで見える

  2. 鮮やかな青・オレンジ・緑は特に要注意

    RGBの発色が得意な色ほど、CMYKに変換するとくすんで見えやすい

  3. CanvaはデフォルトでRGB。印刷用CMYKはPro機能

    無料プランでは変換結果を事前に確認しておくのが唯一の対策

  4. 入稿前に自分でCMYKに変換して確認する

    丸投げ変換より手動変換のほうが仕上がりをコントロールできる

「データ上ではきれいだったのに」という後悔は、ちょっとした事前確認でほぼ防げます。具体的な変換の手順については「RGB→CMYKの変換方法｜Illustratorとツールで確実に変換する」で解説しています。
