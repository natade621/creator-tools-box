JPEGとPNGとPDFって何が違う？デザインで使うデータ形式を整理する

2026年4月1日

目次

  1. まず「ラスター」と「ベクター」の違いを知る
  2. よく使うファイル形式まとめ
  3. 「どれで保存すればいい？」早見表

「PNGで送ってください」「PDF入稿でお願いします」——デザインの仕事をしていると、いろいろなファイル形式が出てくる。最初は「なんとなく用途が違うらしい」くらいの理解でも困らないけど、知らないまま使い続けると、ある日「なぜJPEGじゃダメなんですか？」と聞かれて答えられない場面が来る。

今回は実務でよく使うデータ形式を、初心者向けにまとめて整理しておく。

まず「ラスター」と「ベクター」の違いを知る

データ形式を理解する前に、画像の種類を2つに分けて理解しておく必要がある。

ラスターはピクセル（点）の集まりで表現する画像。ベクターは数式で描かれた図形で、拡大しても劣化しない。

種類 特徴 代表形式
ラスター拡大すると粗くなる。写真に向いているJPEG・PNG・GIF・TIFF
ベクターどんなサイズでも綺麗。ロゴやイラストに向くAI・SVG・EPS

この違いがわかると、「なぜロゴはAIファイルで管理するのか」「なぜ写真をSVGにできないのか」が自然に理解できる。

NOTE

Photoshopが主にラスター、Illustratorが主にベクターを扱うアプリ、という大きな整理もここからくる。

よく使うファイル形式まとめ

JPEG（.jpg）

写真やグラデーションを含む画像の保存に向いた形式。圧縮して容量を小さくする代わりに、多少画質が落ちる。

  • 背景の透過はできない
  • 保存を繰り返すたびに画質が劣化する（非可逆圧縮）
  • Web・SNSへの画像アップロードはほぼこれでOK

NOTE

「最高品質」で書き出しても、何度も上書き保存すると劣化が積み重なる。マスターデータはAIやPSDで保持し、JPEGは書き出し専用にするのが鉄則。

PNG（.png）

背景を透過できる形式。ロゴ・アイコン・イラストのWeb用書き出しに使うことが多い。

  • 透過（透明な背景）が使える
  • 画質は劣化しない（可逆圧縮）
  • JPEGより容量が大きくなりやすい

「白背景のロゴをJPEGで送ったら背景が白で出てきた」はよくあるトラブル。背景透過が必要なロゴはPNGで書き出す。

PDF（.pdf）

印刷入稿・書類共有・レイアウト確認など、幅広い場面で使われる形式。フォントや配置情報をそのまま保持できる。

  • 環境に関わらず同じレイアウトで表示・印刷できる
  • 印刷会社への入稿はPDFが主流になっている
  • Acrobatなど専用ツールがあればテキストや注釈の編集も可能

NOTE

印刷用PDFはRGB→CMYKへの変換や、トンボ・塗り足し設定を正しくした状態で書き出す必要がある。ただ「PDF保存」するだけでは不十分なことも多い。印刷で色がくすむ原因も参考に。

AI（.ai）

Adobe Illustratorのネイティブ形式。ロゴ・イラスト・印刷物のデータはこの形式で管理するのが基本。

  • ベクターなのでどんなサイズでも綺麗
  • レイヤーや編集情報を保持したまま保存できる
  • AIファイルを開くにはIllustratorが必要

「ロゴのデータをください」と言われたら、まずAIファイルを渡すのが正解。JPEGしか出せないと、相手が拡大できなくて困ることが多い。

SVG（.svg）

Web向けのベクター形式。ロゴ・アイコン・シンプルなイラストをWebに埋め込むときに使われる。

  • 拡大しても綺麗（ベクター）
  • テキストエディタで直接編集できる
  • アニメーションもつけられる

NOTE

複雑な写真やグラデーションはSVGに向かない。シンプルな図形・ロゴ・アイコンのWeb用書き出しに活用しよう。

TIFF（.tif）

印刷・写真レタッチ向けの高品質ラスター形式。圧縮による画質劣化がなく、大判印刷や高解像度の写真データに使われる。

  • 容量は非常に大きくなる
  • プロの印刷現場では今も現役で使われる形式

「どれで保存すればいい？」早見表

用途 おすすめ形式
印刷会社へ入稿PDF（印刷設定あり）
ロゴ・素材の管理AI（マスターデータ）
Web・SNSに載せる写真JPEG
透過が必要なロゴ・アイコン（Web）PNG
Webに埋め込むロゴ・アイコンSVG
高解像度写真の保管TIFF または PSD

データ形式の選択は「用途」で決まる。

  1. ラスターとベクターの違いを理解する

    写真はラスター、ロゴ・イラストはベクターが基本

  2. 印刷入稿はPDF

    CMYKとトンボ設定を忘れずに

  3. ロゴのマスターはAI

    JPEGで管理すると後で困ることが多い

  4. Web透過ありはPNG、なしはJPEG
